# Used Cars Price Analysis 🚗

## 📌 Objective
Analyze used car prices to understand factors affecting resale value.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Jupyter Notebook

## 📊 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization

## 📈 Key Insights
- Older cars have lower resale value
- Mileage significantly affects price
- Diesel cars tend to have higher resale value
